## Общая информация

Лабораторная работа выполнена в одном проекте.
Все задания реализованы в виде отдельных классов, сгруппированных по логическим пакетам.

В программе присутствует единственная точка входа — класс `Main` пакета `ru.gulin.main`.
В главном методе `main` продемонстрирована работа всех классов и методов с использованием дружественного интерфейса.

Все исходные данные вводятся с клавиатуры, для каждого ввода реализована проверка корректности данных.

В каждом классе присутствуют:

* свойства (поля);
* конструкторы;
* метод `toString()`.

## Структура проекта
